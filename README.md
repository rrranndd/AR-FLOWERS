# AR-FLOWERS

Aplikasi Pembelajaran Nama Bunga Berbasis Augmented Reality (AR) untuk Anak Tunarungu

## Fitur

- Deteksi kartu huruf menggunakan Vuforia
- Penyusunan huruf menjadi nama bunga
- Menampilkan objek bunga 3D
- Menu Panduan
- Daftar Bunga
- Navigasi antar scene

## Teknologi

- Unity 6
- Vuforia Engine 11.4.4
- C#
